# 🍔 Sistema de Ventas - Fast Food
Sistema de gestión desarrollado en PHP y MySQL para negocios de comida rápida.

## ✨ Características
- Perfil de admin (Gráficos, Gestión de personal, Gestión de ventas y Gestión de Productos).
- Perfil de vendedor personalizado.
- Menú lateral desplegable (Ventas, Pedidos, Reportes).
- Conexión segura con base de datos `sistemadeventas`.

## 🛠️ Instalación
1. Clonar: `git clone https://github.com/Reivaxlm/Sistema-de-ventas-de-comida-r-pida-.git`
2. Importar `sistemadeventas.sql`.
3. Configurar `connection.php`.
